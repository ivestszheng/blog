# 避免编译

`less`会自动帮我们把代码编译成`css`，但是遇到想要避免编译的情况该怎么做呢？

```less
*{
	margin: 100 * 10px;
	padding: calc(100px + 100);
}
```

编译为

```css
*{
	margin: 1000px;
	padding: calc(200px);
}
```

`css3`中本身就有`calc()`,此时不想要`less`进行编译，可以把代码修改为

```less
*{
	margin: 1000px;
	padding: ~"calc(100px + 100)";
}
```

此时编译为

```css
*{
	margin: 1000px;
	padding: calc(100px + 100);
}
```

`calc()`中的值交由浏览器进行计算
